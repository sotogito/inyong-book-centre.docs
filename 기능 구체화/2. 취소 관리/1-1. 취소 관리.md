
### 주요 기능
- 취소 요청이 들어온 데이터를 처리한다
	- 취소 거부 : 배송 완료
	- 취소 수락 : 취소 완료
- 처리하지 않은 취소 요청 데이터 건 수를 띄운다
- 오늘 들어온 취소 요청 건 수를 띄운다
- 최근 3일의 취소 완료 건 수를 띄운다.
