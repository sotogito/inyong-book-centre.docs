### Cursor에게 전달할 프롬프트
#### 개발 환경 정보 전달 (어차피 파일 열거임)
```
   - 프로젝트 구조
   - 사용 중인 기술 스택
   - 데이터베이스 스키마
   - API 엔드포인트
   - 환경 변수
```
#### 피그마 디자인 구현
```
     - 페이지별 디자인 이미지
     - 반응형 브레이크포인트
     - 사용할 폰트, 색상, 아이콘
     - 애니메이션 요구사항
     - 특별한 인터랙션
```
